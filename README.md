# 📅 Denpamen Schedule Bot 

Discord と Google スプレッドシートを連携させて、ゲーム内イベントなどのスケジュールを直感的に管理・通知するための Discord Bot です。
ボタン操作のコントロールパネルから誰でも簡単にスケジュールの追加・削除ができ、最新のスケジュールをガントチャート画像でお知らせします。



---

## 主な機能

1. **📱 コマンド不要のコントロールパネル**
   - 「追加」「一覧」「削除」などの基本操作を専用チャンネルの「ボタン（固定メッセージ）」から行えます。

2. **📊 Google スプレッドシート連携 & ガントチャート生成**
   - 登録したスケジュールは自動でスプレッドシートに保存されます。
   - スケジュール一覧を表示する際には、見やすい「ガントチャート形式の画像」を自動生成して Discord に投稿します。

3. ** 定期通知 & リマインダー**
   - 指定した日時（例：毎週月曜 09:00）に自動でスケジュールをお知らせします。
   - イベント開始の 15分前・60分前 などのリマインダー通知もサポート。
   - GitHub Actions を活用すれば、自宅PCを付けておかなくても定期通知を送信できます。

4. ** 公式サイト新着検知 (オプション機能)**
   - 1時間ごとに公式サイトの新着記事を自動チェック。
   - 新着があれば Discord へ通知するだけでなく、期間が記載されたイベント記事なら**自動的にスケジュールへ追加**します。

5. **🔁 定期（永遠）イベントの自動生成**
   - 「毎週〇曜日」や「毎月偶数日」に行われる定期イベントを設定ファイルに書いておくだけで、数ヶ月先までの仮スケジュールとして一覧やガントチャートに自動で組み込まれます。

6. **🐦 公式Twitter新着通知 (オプション機能)**
   - 1時間ごとに公式Twitterアカウント（@denpaningen）の新着ツイートを自動チェック。
   - 新着があれば Discord へ通知します（RSSHub 経由で取得、APIキー不要）。

7. **🎬 YouTube新着動画通知 (オプション機能)**
   - 1時間ごとに公式YouTubeチャンネルの新着動画を自動チェック。
   - 新着があれば Discord へ通知します（YouTube RSSフィード経由、APIキー不要）。

## 🛠 必要な環境

- **Python:** 3.11 以上
- **必要なAPI/サービス:** 
  - Discord Bot トークン
  - Google Cloud サービスアカウントキー (Google Sheets / Drive API 用)
  - GitHub アカウント（GitHub Actionsでの定期通知機能を使う場合）

## 🚀 導入・セットアップ方法

この Bot を自分の Discord サーバーに導入するには、いくつかのサービスで初期設定を行う必要があります。

詳細な導入手順（スクショを用いたボットの作成方法から、起動・自動化設定まで）は、以下の専用ガイドをご確認ください。

👉 **[SETUP_GUIDE.md](./SETUP_GUIDE.md) （詳細セットアップガイド）**

### クイックスタート概要 (ローカル起動)
1. `SETUP_GUIDE.md` を参考に `.env` ファイルと `service_account.json` を準備
2. 必要なライブラリのインストール: `pip install -r requirements.txt`
3. 起動: `python src/bot.py`
